What's different from v3

  • Onboarding kept. First-launch, downloading, ready — all back, in the v4 shell.
  • No tabs. Settings lives behind a gear icon in the masthead.
  • Editor = document. Title in head + pink ● when modified + Save & read affordance + audio · stale stamp.
  • Audio paused by default. No autoplay on mount; the play button is the only way to start.
  • Sidebar holds the full library with inline delete. Storage breakdown moves into Settings.
